Return Journey Ticket

A Return Journey Ticket allows you to travel to your destination and return to your starting point on the same day. It is convenient for passengers who make a round trip within a day.

Features:
  • Valid for two trips: one onward and one return journey.
  • Both trips must be completed on the same day.
  • Non-transferable and cannot be reused once both journeys are completed.
  • Available at ticket counters and vending machines.
How to Use:
  1. Buy the Return Journey Ticket from a Metro station counter or vending machine.
  2. Use it at the entry gate for your onward trip.
  3. After reaching your destination, use it again at the exit gate.
  4. For the return trip, repeat the same process.
